# Tasks-notifications

Папка содержит атомарные задачи для реализации системы уведомлений в ToDowka Web Client согласно плану v2.1.

## Структура

- `00-guide.md` - Руководство по выполнению задач
- `L0/` - Базовый уровень (5 задач, ~1.5 часа)
- `L1/` - Средний уровень (10 задач, ~5.5 часов)
- `L2/` - Сложный уровень (7 задач, ~7.5 часов)
- `L3/` - Тестирование и документация (4 задачи, ~5.5 часов)

## Общий обзор

Всего **26 атомарных задач** на **~20 часов** разработки.

## Ключевые особенности

### SSE Primary + Polling Fallback
- SSE как основной метод (реальное время)
- Автоматическое переключение на polling при timeout/error (30 сек)
- Периодические попытки восстановления SSE (каждые 60 сек)
- Polling через React Query (единственный механизм)

### Интеграция с React Query
- SSE messages → invalidateQueries
- UI обновляется автоматически
- refetchInterval: false по умолчанию
- Оптимистические обновления

### Компоненты
- NotificationItem - отдельное уведомление
- NotificationList - список уведомлений
- NotificationCenter - dropdown в шапке
- ConnectionStatus - online/offline индикатор
- /notifications - страница всех уведомлений
- /settings - страница с Health Status

### Тестирование
- Unit тесты для компонентов (Jest + RTL)
- Тесты для NotificationStream (EventSource mock)
- Полное покрытие ключевого функционала
